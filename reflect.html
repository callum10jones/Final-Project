Introduction:
My first project walked me through the fundamentals of graphic design, principles of typographic design and how to create imaginative work that is fascinating, dynamic, and thoughtful. Using the fundamentals of HTML and CSS learned in class, I assembled a static website to present the graphical, typographic, and UX foundations, along with short reflection paragraphs to accompany each task. I built the website from the ground up using Visual Studio Code and used version control in Github Desktop to keep track of my code; I also used GitHub to publish and host the static website.
